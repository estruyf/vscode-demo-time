# Project Overview
- Purpose: Demo Time VS Code extension to script and present coding demos and slides inside VS Code.
- Core concept: Demos defined in .demo/demo.json or .demo/demo.yaml with scenes/moves (actions like open, highlight, snippet, slide).
- Key areas: apps/vscode-extension (extension), apps/webviews (UI), packages/common (shared logic), docs (documentation site).
- Repo is an Nx monorepo with workspaces apps/*, packages/*, docs.
