# Pro Features

Demo Time offers Pro features for GitHub Sponsors. These advanced capabilities enhance your presentation and demo experience while supporting the continued development of the extension.

## Accessing Pro Features

1. Open the Pro Features view:
   - From the Command Palette: `Demo Time: Pro Features`
   - From the Acts & Scenes view: Click the "Pro Features" menu item
   - From the Resources panel: Click "Get PRO Features"

2. Authenticate with GitHub to unlock Pro features (if not already authenticated)

3. If you're a GitHub Sponsor, all Pro features will be automatically unlocked

## Pro Features Include

### Presentation Analytics
Track your demo performance with detailed analytics including:
- Timing and duration for each step
- Navigation patterns and flow
- Engagement metrics
- Session export for review and improvement

Access via: `Demo Time: Open Analytics Dashboard`

### Advanced Export Options
- Export slides to PDF with custom templates
- Custom branding and styling
- High-quality output for sharing

### Priority Support
- Faster response times
- Dedicated support channels
- Direct feedback to the development team

### Exclusive Features Access
- Early access to experimental features
- Beta releases and new capabilities
- Help shape the future of Demo Time

## How to Become a Sponsor

1. Click the "Authenticate with GitHub" button in the Pro Features view
2. Or visit: [GitHub Sponsors](https://github.com/sponsors/estruyf)
3. After sponsoring, authenticate in VS Code to unlock Pro features
