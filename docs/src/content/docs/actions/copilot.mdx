GitHub Copilot actions allow you to interact with GitHub Copilot directly from your Demo Time steps. You can open GitHub Copilot chat, start new chats, ask questions, and moreâ€”all as part of your scripted demo.

<Aside type="note" title="Samples">
  You can find Copilot actions samples on GitHub:
  <a href="https://github.com/estruyf/vscode-demo-time/tree/main/.demo">.demo/copilot-actions.json</a>
</Aside>

## Open Copilot chat

To open the Copilot chat panel, use the `openCopilotChat` action.

<Tabs syncKey="copilot-open-chat">
<TabItem label="JSON">

```json
{
  "action": "openCopilotChat"
}
```

</TabItem>
<TabItem label="YAML">

```yaml
action: openCopilotChat
```

</TabItem>
</Tabs>

## Ask a question in Copilot chat

To ask a question, use the `askCopilotChat` action. The `message` property is optional; if omitted, the chat will open without sending a message.

<Tabs syncKey="copilot-ask-chat">
<TabItem label="JSON">

```json
{
  "action": "askCopilotChat"
}

{
  "action": "askCopilotChat",
  "message": "Write a function that returns 'Hello from Demo Time'."
}
```

</TabItem>
<TabItem label="YAML">

```yaml
action: askCopilotChat
```

```yaml
action: askCopilotChat
message: "Write a function that returns 'Hello from Demo Time'."
```

</TabItem>
</Tabs>

## Start a custom chat with Copilot

To start a chat with a custom mode, use the `customCopilotChat` action.

Define the `mode` property to specify the chat mode you want to use.

The `message` property is optional.

<Tabs syncKey="copilot-custom-chat">
<TabItem label="JSON">

```json
{
  "action": "customCopilotChat",
  "mode": "Beast Mode 3.1"
}

{
  "action": "customCopilotChat",
  "mode": "Beast Mode 3.1",
  "message": "Write a function that returns 'Hello from Demo Time'."
}
```

</TabItem>
<TabItem label="YAML">

```yaml
action: customCopilotChat
mode: "Beast Mode 3.1"
```

```yaml
action: customCopilotChat
mode: "Beast Mode 3.1"
message: "Write a function that returns 'Hello from Demo Time'."
```

</TabItem>
</Tabs>

## Cancel Copilot chat

Use the `cancelCopilotChat` action to cancel an ongoing GitHub Copilot request while keeping the chat panel open. This is useful when a previous request is still running and you want to abort it before continuing with the demo.

<Tabs syncKey="copilot-cancel-chat">
<TabItem label="JSON">

```json
{
  "action": "cancelCopilotChat"
}
```

</TabItem>
<TabItem label="YAML">

```yaml
action: cancelCopilotChat
```

</TabItem>
</Tabs>
