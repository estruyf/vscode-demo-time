Time actions are used to perform actions related to time in the editor. You can use these actions to wait for a specific time, wait for user input, and more.

<Aside type="note" title='Samples'>
  You can find time actions samples on GitHub: 
  <GitHubFileOpener filePath=".demo/4.time-actions.json" />
</Aside>

## Wait for a specific time

Wait for a specific time before executing the next action.

<Tabs syncKey="wait-for-time">
<TabItem label="JSON">

```json
{
  "action": "waitForTimeout",
  "timeout": "<timeout in milliseconds>"
}
```

</TabItem>
<TabItem label="YAML">

```yaml
action: waitForTimeout
timeout: <timeout in milliseconds>
```

</TabItem>
</Tabs>

## Wait for user input

Wait until the user presses a key.

<Tabs syncKey="wait-for-input">
<TabItem label="JSON">

```json
{
  "action": "waitForInput"
}
```

</TabItem>
<TabItem label="YAML">

```yaml
action: waitForInput
```

</TabItem>
</Tabs>
