There are many ways you can contribute to Demo Time. 

You can help us by:

- Reporting bugs
- Suggesting new features
- Improving the documentation
- Improving the code
- Spreading the word ❤️

## Getting started

If you want to contribute to the development of Demo Time, you can start by forking the [vscode-demo-time - GitHub repository](https://github.com/estruyf/vscode-demo-time).

Once forked, follow these steps to get started:

- Clone the repository to your local machine
- Install the dependencies by running `npm install`
- Run `npm run watch`
- Press `F5` to start the extension in a new Visual Studio Code window
- Start coding

## Reporting bugs and suggesting features

If you find a bug or have an idea for a feature, you can help us by creating a new issue in the [vscode-demo-time - GitHub repository](https://github.com/estruyf/vscode-demo-time/issues).

## Improving the documentation

If you find a typo or want to improve the documentation, you can do so by editing the documentation directly on GitHub.

The source of the documentation can be found in the [demo-time-docs - GitHub repository](https://github.com/estruyf/demo-time-docs).

## Spreading the word

If you like Demo Time, you can help us by spreading the word. Share it with your friends, colleagues, and on social media.

If you have articles, videos, projects, or anything else related to Demo Time feel free to share it with us. We can add it to the list of projects or showcases on the website.
